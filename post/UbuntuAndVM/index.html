但是安装一些库的时候也遇到了一些问题

  • 使用下面的方法那幢nodejs版本低
    sudo apt install nodejs
    //此方式安装显示Nodejs v8.10.0

  • 解决方式：安装稳定版本
    sudo apt-get install npm
    sudo npm insatll -g n
    sudo n stable
    //安装nodejs稳定版本 V16.14.2

  • 安装truffle一直报错，在最前面加上sudo授权就解决了
    sudo npm install -g truffle

  • 打开图形化界面
    nautilus
