GitHub Pages绑定自定义域名

· 2022-04-13 · # GitHub Pages # 个人域名 # 技术分享 # 网站配置

GitHub Pages可以配置网站，省去数据库管理域名备案等问题，但网址冗长不易记忆且不够炫酷👀
本文介绍内容，基于GitHub Pages构建你的个性化域名网站

1.购买个人域名，1美刀起步

https://www.namesilo.com/
输入你喜欢的域名关键词并SEARCH(搜索)，点击Show Advanced Search有更多的域名后缀


目前都在做特价(一年特价，后面原价续费)，考虑到后期续费，我选择的是top后缀域名
目前还有特价码savingone(节省1美元)，如果失效请自行百度

2.确认GitHub仓库后台IP

进入你的GitHub仓库Settings-->Pages(请自行开启)-->Custom domain-->输入购买的域名并Save-->Learn more

右上角搜索"配置 apex 域"并定位至A记录IP处备用

3.Github IP与绑定Namsile域名绑定

在Namesilo官网进行以下操作
My account-->Manage My Domains-->在对应域名后点击蓝色球体(如图)

按步骤1清空目前所有IP，重复2-3-4步骤，把GitHub的四个IP添加进入

恭喜你已完成所有步骤，不过您需要等待1-24小时，你的域名才能生效

如果您设置成功，请我喝杯咖啡~
